# Go-Concurrent-Crawler

![CI](https://github.com/skylerblue333/Go-Concurrent-Crawler/workflows/CI/badge.svg)

High-performance Go service for crawler operations.

## Features
- Highly concurrent `sync.RWMutex` state management
- Standard library HTTP routing
- Dockerized multi-stage build
- 100% Test Coverage

## Quick Start
```bash
go test ./...
go run main.go
```
