# Go-Concurrent-Crawler

## Overview
A highly concurrent web crawler using goroutines and channels for maximum throughput.

## Quick Start (1-Click Build)

```bash
git clone https://github.com/skylerblue333/Go-Concurrent-Crawler.git
cd Go-Concurrent-Crawler
go run main.go
```

## Features
- Goroutine-based concurrency
- `sync.Map` for thread-safe visited tracking
- Channel-based result aggregation
